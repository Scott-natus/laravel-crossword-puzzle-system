## 이미지 설정 안내

### 메인 배너 이미지
조감도 이미지를 다음 경로에 업로드해주세요:
```

```

- **크기**: 1920x1080px 이상
- **형식**: JPG, PNG
- **파일명**: uljin-apartment-aerial.jpg
